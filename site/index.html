html 基本标签

  • 语义 HTML 标签
    1. 可能有助于页面的搜索引擎优化排名
    2. 帮助计算机更好的理解页面结构
    3. 帮助人类更好的理解页面结构
  • &字符实体引用
    1. 引用字符实体的目的：允许浏览器显示某些字符。否则浏览器会将其理解为 HTML 代码的一部分，而不会显示给用户。
  • 列表(无序<ul>,有序<ol>)
    1. ©版权符号的实体标签
    2.  确保文档中的三个单词无论浏览器窗口怎样变化始终在同一行(实体和单词之间不留空格)
  • 链接标签
    1. 跳转到另一个 html 文件中
    2. 跳转到别的网页
    3. 在同一个网页中跳转（ 段落、页眉、页脚。在  link-example.html  中演示）
    4. 将 <target='_blank'> 属性列为 <a >标记的一部分可以强制浏览器在新窗口或标签页中打开链接
