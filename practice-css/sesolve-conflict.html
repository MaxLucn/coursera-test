将所有 css 样式放到 html 页面外部的单独文件是个好主意，因为这样可以在多个页面中共享相同的样式

当对目标指定了相同的属性且产生冲突时，原始优先规则就会生效：最后一个声明有效（由于 html 页面是从上往下的，所以很好确定哪个才是最后一个声明(origin)）

当对目标指定了不同的属性时就不存在，这时候它们的属性就会同时生效，即：声明合并(declarations Merge)

css 规则越具体优先级越高，这里的具体指的是直接指定或者给标签给定属性的情况下，它们优先级为

  1. 直接给某标签指定 style
  2. 有 ID 属性的标签
  3. 定义了 类(class)的标签
  4. 定义了伪类的标签
  5. 最后是所选组合中使用的元素数量
